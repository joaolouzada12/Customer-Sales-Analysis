# Relational Sales Analysis

This is a more advanced data analysis project using Python and Pandas.

The data is simulated and organized into three tables:
- clients
- products
- sales

The goal is to practice working with relational data, where it's necessary to connect tables using IDs before doing the analysis.

## What I did

- generated the datasets
- organized the data in multiple tables
- prepared the structure for joins (merge)
- followed a basic data analysis workflow

## Technologies

- Python
- Pandas
- Matplotlib
